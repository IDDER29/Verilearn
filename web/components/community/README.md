# `web/components/community` — community thread controls

Interactive controls for the community threads (`/community/*`). Community is an R2/R3 surface — the data model is lightly seeded — but these controls are real, keyboard-operable widgets (COMM-16).

| File | What it is |
|---|---|
| `ThreadVote.tsx` | Up/down vote control on a thread/reply. |
| `ShareButton.tsx` | Share a thread. |
