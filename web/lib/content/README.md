# `web/lib/content` — static curated content

Hand-authored content that isn't user data and isn't generated — it ships with the app.

| File | What it is |
|---|---|
| **`exampleTopics.ts`** | A fixed, curated list of example topics used by the Dashboard's "Learn next" row (HOME-21) and onboarding prompts. Honestly non-personalized — a starting-point catalog, not a recommendation engine. Topics already owned by the learner are filtered out at the call site. |

If more static content appears (curated source lists, canned help text, etc.), it belongs here.
