The one thing to fix

Mixed icon languages

Everything uses clean line icons (Tabler) — except Sources, which uses emoji (📘 🔬 📄 🌐). It's the one place the visual language breaks. Either commit to line icons for source types, or make emoji an intentional, consistent accent. Small fix, noticeably more finished.
