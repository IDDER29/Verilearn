Principles

What makes these work

01
Never a dead end
Every empty state names the next step or reassures that empty is the good state.
02
One action, not five
A blank screen is the one place to be ruthless about a single primary CTA.
03
Teach the feature
First-empty is a chance to explain how the thing fills up — gaps, sources, reviews.
04
Reward, don't scold
"All caught up" and "no conflicts" are wins — style them as such, never as errors.
