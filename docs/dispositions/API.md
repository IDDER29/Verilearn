### Integrations, API, Webhooks, SSO & LTI (API)

This domain is the external/federated surface (public REST/GraphQL, API keys, OAuth, signed webhooks, SSO, SCIM, LTI, verify endpoint/widget, sandbox, versioning). The codebase has **no HTTP API surface**: `app/` contains no `route.ts` / `api/` handlers, and there is no webhook, OAuth, SAML/OIDC, SCIM, or LTI code anywhere in `web/`. What *is* built are the internal domain primitives any future API would sit on: fail-closed certificate verify (`lib/domain/certificates.ts`), the five-state trust ledger + coverage matrix (`lib/domain/trust.ts`, `lib/services/sources.ts`), the four honest signals (`lib/domain/signals.ts`), and the epistemic firewall in the capability layer (`lib/domain/rbac.ts`, `lib/domain/trust.ts`). Classifications reflect that reality.
